Welcome, students! In today’s tutorial, we shall learn to add and subtract rational algebraic expressions with like denominators.

Rational expressions are just fractions, but instead of numbers, they include variables. This might sound tricky, but once you get a hold of how to proceed, you’ll find it easy.

Please be guided with the following procedures when adding or subtracting rational algebraic expressions with like denominators:

1. Add (or subtract) the numerators.

2. Copy the common denominator.

3. Simplify the result.

Let’s get started with a quick review.

A rational algebraic expression is a fraction whose numerator and denominator are both polynomials.

Whenever you add or subtract expressions which contain like denominators, you don’t have to worry about the denominator, just add the numerators instead.

For instance, given the example: 1/2x + 1/2x.

Notice that the denominators are the same, so we’ll need to focus on adding the numerators.

We add the numerators like this: (1 + 1)/2x = 2/2x.

Simplify the numerators by combining like terms: 2/2x ÷ 2/2.

Which gives us: 1/x.
